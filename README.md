# The Tree Map - NYC
By Sofía Molina // [View live project](https://sofiams.com/Tree-Map/) 
### 

Interactive visualization of New York City’s street tree canopy, built using official city forestry data and styled like a clean architectural site plan. It allows users to explore individual trees across NYC’s boroughs, complete with details on species, health, diameter, coordinates, and Google Street View links, while offering features like 3D isometric toggles, species filtering, and custom layer controls for parks, water, and buildings.
